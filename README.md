# GMAP MicroServices

Build Project
``` 
mvn clean install
```
## bidservice

```
cd bidservice
mvn spring-boot:run
```
## userservice

```
cd userservice
mvn spring-boot:run

```

Use BID.postman_collection.json for trying out various request.
a) Teanant Creation (bidservice)
b) Bid Creation (bidservice)
c) Update Bid (bidservice)
d) Retrive Tenant (userservice)
